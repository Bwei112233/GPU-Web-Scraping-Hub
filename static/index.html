Welcome to the WebScraping Hub

Use the drop down menu below to choose which website to collect data from major retailers. Your data will be sent to you in a .csv file. Data is collected using python and cleaned using Beautiful Soup. Retailers might block access to their websites if the web scraper is run too frequently. If this happens wait for the cooldown to expire or click here to see sample data. If you have any new features to suggest of bugs to report, feel free to contact me on Github. Enjoy!
